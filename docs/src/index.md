# SolarFunctions.jl

*Julia functions for computing Sun-related quantites.*

`SolarFunctions.jl` is a collection of pure functions which compute useful quantites for Sun-related calculations, such as [irradiance](https://en.wikipedia.org/wiki/Irradiance) and [solar position](https://en.wikipedia.org/wiki/Position_of_the_Sun). 
A detailed list of the implemented function is provided in the [API docs](api.md).   

Install `SolarFunctions.jl` from the Julia REPL

```julia
add ] https://github.com/nicomignoni/SolarFunctions.jl.git
```
